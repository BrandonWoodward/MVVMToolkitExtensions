Namespace MVVMToolkitExtensions.WPF.Interfaces

Interfaces

IDialogAware

Provides methods to handle the lifecycle of a dialog window. Implementing this interface allows a ViewModel to control and respond to the opening and closing events of its associated dialog.

IDialogService

Service used to show dialogs and pass parameters to their ViewModels.

INavigationAware

Provides methods to handle the navigation lifecycle of views or pages. Classes that implement this interface can receive notifications when they are navigated to or from.

IRegionManager

Provides a contract for services that manage view-based navigation within regions defined by RegionControl.
